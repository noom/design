Coach Dashboard Style Guide

These guidelines are intended to help standardize mockups and design for Noom's core application: Noom Coach. This does not include each individual element or interaction in the app, but rather aims to provide the design patterns which make up the core Noom Coach Features.

General

The Noom design system is comprised of a series of page design elements and modules which work together to shape the Noom experience. Use the rules outlined in this design system to create new features or functionality to ensure a consistent user experience.
