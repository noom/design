Web Component Guidelines

This section of the design guides aims to establish a development framework that helps create responsive layouts quickly and efficiently. The color and aesthetic suggestions are in line with the published brand guide. If you notice anything is wrong or needs to be changed, you can report an issue by bothering the DUX team ({{ site.email }}).

Layout

The website layout follows a twelve column grid system.

Typography

Typeface

Roboto is our typeface. We use it mostly in the lighter weights — light, regular and medium. The font color, size and weight depends on each use case, and is outlined below. Note that the default color for text colors is grey (#4D4D4D).
