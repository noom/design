Marketing Site Style Guide

At Noom, we believe in creating a harmonious experience for our users. This is accomplished by applying the same solutions to challenging problems consistently across the both the mobile as well as the web application. This design style guide aims to bring consistency and reusability across different areas of the web platform. The color and aesthetic suggestions are in line with the published brand guide. A printed version of the brand guide can be referenced by bothering the DUX team ({{ site.email }}).

Layout

The website layout follows a twelve column grid system.

Typography

Main typeface

Roboto is our typeface. We use it mostly in the lighter weights — light, regular and medium. The font color, size and weight depends on each use case, and is outlined below. Note that the default color for text colors is grey (#4D4D4D).
