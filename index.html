Colors

The color palette consists of 18 colors, each with 6 shades. The various shades are defined numerically with 1 being the lightest and 6 being the darkest.

Brand Color Palette

Primary Color Palette

The primary color palette consists of 10 colors, each with 4 shades. It is meant to be used for design of all in app product features. For specific details on the use of each color within the app, see the Interface Guidelines

Secondary Color Palette

The Secondary color palette consists of 18 colors each with 6 shades. It includes an extension of the priamry palette shades as well as additional colors. It is meant to be used for illustration and the website design. For more details on the color use in the website design, see the Website Guidelines
